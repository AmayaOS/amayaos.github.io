AmayaOS LiveCD

This guide will help you easily burn AmayaOS on a CD (or DVD).

Get AmayaOS

First of all we have to get a copy of the AmayaOS Image. We can download one of the precompiled images or compile the AmayaOS source code ourselves (for advanced users only).

Dependencies

There are many different ways to burn the ISO file on the CD but the easiest way is using one tool like k3b (In GNU/Linux). Of course, you can do this with any other burning tool.

When the CD is burned you have just to restart your computer and change the BIOS Boot Sequence to allow the CD to boot.
