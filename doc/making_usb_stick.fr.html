AmayaOS LiveUSB

Cette notice vous aidera à graver AmayaOS sur une clé USB ou d'autres médias amovibles similaires telle qu'une carte mémoire SD, de façon simple et dans quelques instants.

Obtenir AmayaOS

D'abord il faut obtenir une copie d'une image d'AmayaOS. Vous pouvez télécharger une des images précompilées ou bien compiler d'après le code source une image d'installation (seulement pour les utilisateurs avancés).

Dépendances

Pour graver l'image ISO sur le USB (ou d'autres médias amovibles), on peut utiliser la commande UNIX® dd (duplicate disk). Cet outil doit être installé par défaut sur toutes les distributions GNU/Linux, derivées de BSD (FreeBSD, OpenBSD, etc) ou HaikuOS. Au cas improbable où dd ne soit pas installée sur votre système vous devrez l'installer.

En gravant l'image

Pour graver l'image, il faut exécuter dd de la manière indiquée dans la suite, étant « amayaos-usb.iso » le nom de l'image ISO et « /dev/sdX » le répertoire identifiant le média amovible (/dev/sda, /dev/sdb, etc). Ce répertoire peut utiliser d'autres noms si on utilise d'autres systèmes d'exploitation ou types de média (par example, une carte SD peut se nommer /dev/mmcblk0).

user@domain$ dd if=amayaos-usb.iso of=/dev/sdX

*Note : Sous GNU/Linux vous aurez très probablement besoin d'exécuter la commande avec les privilèges d'administrateur (root).

Une fois l'image soit gravée sur le USB (cette opération peut durer quelques secondes) on doit tout simplement redémarrer notre ordinateur et configurer la séquence de démarrage pour changer la priorité et démarrer depuis le USB (ou carte mémoire SD).
