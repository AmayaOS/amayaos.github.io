AmayaOS LiveCD

Ce document vous aidera à graver AmayaOS sur un CD (ou DVD) de façon simple et dans quelques instants.

Obtenir AmayaOS

D'abord il faut obtenir une copie d'une image d'AmayaOS. Vous pouvez télécharger une des images précompilées ou bien compiler d'après le code source une image d'installation (seulement pour les utilisateurs avancés).

Dépendances

Il existe beaucoup d'options pour graver l'image ISO d'AmayaOS sur le CD, mais la manière plus simple est d'utiliser un logiciel de gravure tel que Brasero ou k3b (normalement disponibles dans les référentiels des distributions GNU/Linux). Vous pouvez aussi graver avec d'autres logiciels de votre choix.

Une fois gravée sur le CD (la durée du processus dépend du logiciel et la vitesse choisie de gravure) on peut redémarrer la machine. Après on doit changer la priorité de démarrage dans la BIOS pour démarrer depuis le CD et comme ça notre ordinateur démarrera sous AmayaOS.
