AmayaOS LiveCD

Este documento te ayudará a grabar AmayaOS en un CD (o DVD) de forma sencilla y en tan solo unos pocos minutos.

Obtener AmayaOS

Lo primero de todo es obtener una copia de la imagen de AmayaOS, para ello podemos descargar alguna de las imágenes precompiladas o compilar el código fuente de la imagen de instalación (solo para usuarios avanzados).

Dependencias

Existen muchas formas para grabar la imagen ISO de AmayaOS en el CD, pero la más sencilla es hacerlo utilizando una herramienta como brasero o k3b (ambas son facilmente instalables en distribuciones GNU/Linux). También se puede grabar la imagen con cualquier otra utilidad a elección del usuario.

Una vez grabada la imagen en el CD (la duración del proceso depdende del software utilizado para grabar) simplemente reiniciamos nuestra máquina, cambiamos el orden de arranque en nuestra BIOS para que tenga preferencia el CD y nuestro ordenador arrancará con AmayaOS.
