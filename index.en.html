About AmayaOS

What is AmayaOS?

AmayaOS is an operating system similar to UNIX which is free software and respects your freedom. You can install AmayaOS on your computer without any concerns. (It’s 100% free software). AmayaOS project was launched in 2014 to create a new operating system useful for the x86 pc.

AmayaOS is not GNU/Linux

AmayaOS does not derive from Linux or GNU, and it is not a GNULinux distro. AmayaOS is an operating system which has an independent development based on FreeNOS (Free Niek's Operating System).

AmayaOS follows the UNIX® philosophy

“Write programs that do one thing and do it well. Write programs to work together. Write programs to handle text streams, because that is a universal interface.” – Doug Mcllroy

What are the project goals?

We want to create a new operating system capable of working well in a i586 computer.

What is Free Software?

Free software is the one that contains the 4 different types of freedom included in the philosophy of free software and these types of freedom allow the users to have control over the computer, both individually and in groups. These are the 4 essential freedoms:

  • The freedom to run the program as you wish, for any purpose (freedom 0).

  • The freedom to study how the program works, and change it so it does your computing as you wish (freedom 1). Access to the source code is a precondition for this.

  • The freedom to redistribute copies so you can help your neighbor (freedom 2).

  • The freedom to distribute copies of your modified versions to others (freedom 3). By doing this you can give the whole community a chance to benefit from your changes. Access to the source code is a precondition for this.
