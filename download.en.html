Download AmayaOS

AmayaOS is a free Operating System. Enjoy it! You can get AmayaOS downloading one of the following precompiled images or compiling the source code.

Install Image

It can be burned into a CD, DVD, USB or be used in a virtual machine. This image allows you to install AmayaOS on a real or virtual hard disk.
Live Image

This image executes the full AmayaOS system from a removable device (such as CD, DVD or USB). Also it can be emulated on a virtual machine.

You can get precompiled images of stable versions of AmayaOS in the following links. You can also download development builds which includes the latest code and software, but they often have several bugs. The following guides will help you if you are new in the AmayaOS universe :)
