Código fuente

El código fuente es un conjunto de líneas de texto que contienen las instrucciones que debe seguir el ordenador para realizar una tarea. A partir del código fuente de AmayaOS se pueden generar las imágenes del sistema, así como modificarlo y añadirle nuevas características. El usuario dispone de todos estos derechos ya que AmayaOS está licenciado bajo la GNU GPL v3.

Puedes obtener el código fuente más moderno descargando un archivo comprimido desde el siguiente enlace, o clonando el repositorio Git de AmayaOS:

user@domain$ git clone https://github.com/dalmemail/AmayaOS
